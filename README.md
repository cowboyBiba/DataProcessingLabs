### Лабораторные работы по предмету "Прикладная обработка данных с испоользованием Python"

В репозитории собраны программы лабораторных работ по моему университетскому курсу. Лабы простые, собраны для истории и практики работы с гитом. 
Для проверки работоспособности первой лабы нужен файл, к которому у текущего пользователя не хватит разрешений.

На Linux можно, например, сменить владельца: 

```chown <новый владелец> <путь к файлу>```
